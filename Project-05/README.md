# 😊 Project 05 - Smiley Face Using Matplotlib

## 📌 Overview

This project demonstrates how to create a smiley face using Python and Matplotlib by combining different geometric shapes such as circles and arcs.

---

## 🎯 Objective

To understand how basic plotting and patches in Matplotlib can be used to create simple graphical illustrations.

---

## 🛠 Technologies Used

- Python
- Matplotlib

---

## 📚 Concepts Covered

- Circle Patch
- Arc Patch
- Plot Customization
- Colors
- Figure Design

---

## 📸 Output

A colorful smiley face created completely using Matplotlib.

---

## 🚀 Learning Outcome

This project enhanced my understanding of Matplotlib patches and how geometric shapes can be combined to create creative visual graphics.
