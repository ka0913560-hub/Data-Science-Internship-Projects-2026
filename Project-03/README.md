# 📊 HR Analytics Dashboard

## 📌 Project Overview

This Power BI dashboard provides valuable insights into employee attrition, workforce demographics, salary distribution, and departmental performance. The dashboard helps HR teams identify patterns behind employee turnover and supports data-driven decision making.

---

## 🎯 Objective

To analyze employee attrition and visualize important HR metrics using Power BI.

---

## 📂 Dataset

HR Analytics Dataset

---

## 📊 Dashboard Highlights

- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Experience
- Attrition by Department
- Attrition by Salary Slab
- Attrition by Job Role
- Age Group Distribution
- Attrition by Gender
- Attrition Trend by Experience
- Employee Count by Department

---

## 🛠 Tools Used

- Power BI
- Power Query
- DAX
- CSV Dataset

---

## 📈 Key Insights

- Sales and Administration departments have the highest attrition.
- Employees with lower salary slabs show higher attrition.
- The 26–35 age group has the highest employee count.
- Employee experience significantly impacts attrition trends.
