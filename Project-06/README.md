# 📊 Project 06 - Seaborn Data Visualization

## 📌 Overview

This project demonstrates different statistical data visualizations using the Seaborn library in Python. Various plots were created to understand distributions, relationships, and correlations within datasets.

---

## 🎯 Objective

To learn and implement statistical data visualization using Seaborn for exploratory data analysis (EDA).

---

## 🛠 Technologies Used

- Python
- Seaborn
- Matplotlib
- Pandas

---

## 📊 Visualizations Included

- Line Plot
- Bar Plot
- Count Plot
- Box Plot
- Violin Plot
- Swarm Plot
- Pair Plot
- Heatmap

---

## 📂 Dataset Used

- Tips Dataset (Built-in Seaborn Dataset)

---

## 🚀 Learning Outcome

- Learned how to create statistical visualizations using Seaborn.
- Understood categorical and numerical plots.
- Explored relationships between variables.
- Learned correlation analysis using Heatmaps.
