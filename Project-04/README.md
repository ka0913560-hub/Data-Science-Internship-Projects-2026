# 🏠 Project 04 - Hut Drawing Using Matplotlib

## 📌 Overview

This project demonstrates how to create a colorful hut illustration using Python and Matplotlib. Various geometric shapes such as lines, polygons, and circles were combined to build the complete drawing.

---

## 🎯 Objective

To understand the fundamentals of Matplotlib plotting by creating a simple hut using coordinates, lines, filled polygons, and patches.

---

## 🛠 Technologies Used

- Python
- Matplotlib

---

## 📚 Concepts Covered

- Line Plot
- Polygon Drawing
- Color Filling
- Circle Patch
- Coordinate System
- Figure Customization

---

## 📸 Output

A colorful hut created entirely using Matplotlib.

---

## 🚀 Learning Outcome

This project improved my understanding of plotting coordinates, drawing geometric shapes, and using Matplotlib creatively for custom visualizations.
