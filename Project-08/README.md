# 🚗 Car Price Prediction using Linear Regression

## 📌 Project Overview

This project predicts the selling price of a used car using Machine Learning (Linear Regression).

A Streamlit web application is developed where users can enter car details and instantly get the estimated selling price.

---

## 📂 Dataset

Dataset: CarDekho Used Car Dataset

Features used:

- Year
- KM Driven
- Fuel Type
- Seller Type
- Transmission
- Owner
- Mileage
- Engine
- Max Power
- Seats

Target:

- Selling Price

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Streamlit
- Pickle

---

## 📊 Machine Learning Model

Algorithm Used:

- Linear Regression

Evaluation Metrics:

- R² Score
- Mean Squared Error (MSE)
