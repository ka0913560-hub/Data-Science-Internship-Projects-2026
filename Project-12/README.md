# 🩺 Diabetes Prediction using Random Forest Regression

This project predicts diabetes probability using the Random Forest Regression algorithm. A Streamlit web application is used to take user input and display the prediction score.

---

## 📌 Project Overview

This machine learning project is built using the Diabetes dataset. The model is trained with the Random Forest Regressor algorithm and deployed using Streamlit for an interactive user interface.

---

## 🚀 Features

- Predicts diabetes probability based on user input.
- Interactive Streamlit web application.
- Trained using Random Forest Regression.
- Model saved using Joblib.
- Simple and user-friendly interface.
