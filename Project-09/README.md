# 🌸 Iris Flower Prediction using Logistic Regression

A Machine Learning classification project that predicts the Iris flower species using Logistic Regression. The application is built with Streamlit and allows users to enter flower measurements to get instant predictions.

---

## 📌 Project Overview

This project classifies Iris flowers into two species:

- 🌼 Versicolor
- 🌸 Virginica

The prediction is based on the following flower measurements:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

## 🚀 Features

- Data Cleaning & Preprocessing

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/Iris-Flower-Prediction.git
```

Move into the project folder

```bash
cd Iris-Flower-Prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
