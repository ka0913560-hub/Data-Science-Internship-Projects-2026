# 🎬 Movie Recommendation System

A simple Movie Recommendation System built using **Python, Streamlit, Pandas, and Scikit-learn**.

Users can select a movie **genre**, and the system recommends movies from that genre.

---

## 🚀 Features

- 🎭 Select any movie genre
- 🎬 Get movie recommendations instantly
- 📋 Displays Top 10 movies first
- 📂 "Show More" option to view all movies
- 🖥️ Simple and user-friendly Streamlit interface

---

## 🛠️ Technologies Used

- Python
- Pandas
- Streamlit
- Scikit-learn
- LabelEncoder

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Movie-Recommendation-System.git
```

Move into the project folder:

```bash
cd Movie-Recommendation-System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
